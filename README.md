# Project Recursion

This project is part of the Odin Project curriculum with the main objective of demonstrating the use of recursion in programming through the following exercises.

## Fibonacci Sequence

In this first part of the project, a method called ```#fibs(n)``` was implemented which returns an array containing ```n``` numbers of the Fibonacci Sequence. A separate method was implemented with the same objective but in a recursive way.

## Merge Sort

```merge_sort(array)``` is an algorithm that divides and sorts an algorithm recursively as follows:
    1. If the array's length is less than 2 (contains one element or empty), return the array
    2. Split the array into two halves.
    3. Sort the left half.
    4. Sort the right half.
    5. Merge both halves of the array and return the result
